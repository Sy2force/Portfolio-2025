## 🛠️ Installation et Développement

### Prérequis

- Node.js 18+ et npm
- Git

### Installation

1. **Cloner le repository**

```bash
git clone https://github.com/shayacoca/portfolio-2025.git
cd Portfolio-Shay
```

2. **Installer les dépendances**

```bash
# Frontend
cd frontend && npm install
```

3. **Développement**

```bash
# Démarrer le frontend (port 3000+)
npm run dev
```

### Build et Production

```bash
# Build (100% fonctionnel)
npm run build

# Lint (aucune erreur)
npm run lint
```
